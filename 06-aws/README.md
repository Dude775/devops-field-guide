# Module 07: AWS - Cloud Fundamentals

> **Status**: Upcoming
> **Prerequisites**: [Module 03 - Networking](../03-networking/), [Module 05 - Git & GitHub](../05-git-github/)

## Overview

AWS is the dominant cloud platform. This module covers core services (EC2, S3, VPC, IAM), cloud architecture patterns, and the operational mindset needed to manage production infrastructure in the cloud.

## What Will Be Covered

- AWS account setup and IAM
- EC2 instances and networking (VPC)
- S3 storage and access patterns
- Security groups and network ACLs
- Cost management basics
- AWS CLI

Content will be added as the course progresses.
